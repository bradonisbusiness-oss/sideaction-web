# SideAction Project Memory

## Purpose
SideAction is a betting accountability and reputation app.

Core idea:
Users post bets or challenges, track progress, settle outcomes, and build reputation.

## Current stack
Code repo: bradonisbusiness-oss/sideaction-web  
Hosting: Vercel  
Production branch: master  
Working branch: dev  
Production domain: sideactionapp.com

## Deployment workflow
Work happens on dev  
Changes go to production through pull requests into master  
Vercel deploys automatically from the repo

## Current homepage
Main file: index.html  
Logo file: logo.png

## Rules
Do not push directly to master  
Do not rename core files without approval  
Prefer small commits
